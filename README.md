# Sistem Pendaftaran Online STT Baptis Di Papua
Website pendaftaran mahasiswa baru berbasis HTML, Tailwind CSS, dan Alpine.js.
Proyek ini menyediakan alur untuk pengunjung (calon mahasiswa) dan admin kampus dalam satu paket halaman frontend.

## Fitur Utama
- Halaman login terpisah di `index.html`.
- Formulir pendaftaran multi-step di `daftar.html`.
- Dashboard admin kampus di `admin.html`.
- Tabel data admin dengan:
  - pencarian (search),
  - filter dropdown,
  - pagination,
  - tombol CRUD (tambah, edit, hapus),
  - simulasi download Excel dan PDF.
- Navigasi atas (topbar), logo, dan favicon terpasang.
- Dukungan role:
  - Pengunjung: akses langsung formulir pendaftaran.
  - Admin Kampus: akses dashboard data pendaftaran.

## Teknologi yang Digunakan
- HTML5
- Tailwind CSS (CDN)
- Alpine.js (CDN)
- Font Awesome Icons
- Google Fonts (Plus Jakarta Sans)
